Fusão de metais

Pergunta 1: O ferro é um metal amplamente utilizado em construções e na fabricação de veículos. Qual é a quantidade de calor necessária para aquecer 500 g de ferro a partir da temperatura ambiente (25°C) até sua temperatura de fusão (aproximadamente 1.538°C)? Utilize a fórmula Q = mcΔT, onde a capacidade térmica do ferro é de cerca de 0,45 J/g°C. Dica: comece calculando a variação de temperatura.

Pergunta 2: Uma barra de alumínio com massa de 1.000 gramas é aquecida até sua temperatura de fusão de aproximadamente 660,3°C. Se a barra estava inicialmente a 20°C, quanto calor foi fornecido para realizar essa fusão? Utilize a fórmula Q = mL, onde o calor latente de fusão do alumínio é de cerca de 397 J/g. Dica: comece calculando a quantidade de calor necessária para aquecer o alumínio até a temperatura de fusão

Resposta 1 : Primeiro, calcule a variação de temperatura:
ΔT = 1.538°C - 25°C = 1.513°C
Agora, utilize a fórmula Q = mcΔT:
Q = 500g * 0,45 J/g°C * 1.513°C = 340.425 J

Respostas

Resposta 2: Primeiro, calcule a quantidade de calor necessária para aquecer o alumínio até sua temperatura de fusão:
Q1 = mcΔT = 1000 g* 0,9 J/g°C * (660,3°C - 20°C) = 532.629 J
Agora, calcule aquantidade de calor necessária para a fusão do alumínio:
Q2 = mL =1000 g * 397 J/g = 397.000 J
A quantidade total de calor fornecida é a soma de Q1 e Q2:
Q total = Q1 + Q2 = 532.629 J + 397.000 J = 929.629 J

Fusão de gelo

Pergunta 1: Você tem um cubo de gelo de 50 gramas a -10°C. Quanta energia calorífica é necessária para elevar a temperatura do gelo até 0°C? Utilize a fórmula Q = mcΔT, onde a capacidade térmica do gelo é de cerca de 2,1 J/g°C.

Pergunta 2: Uma pessoa tem uma placa de gelo de 200 gramas a -20°C e deseja transformá-la completamente em água líquida a 10°C. Qual a quantidade total de calor necessária para realizar essa transformação? Utilize a fórmula Q = mL para o processo de fusão e Q = mcΔT para aquecer a água líquida até 10°C. O calor latente da fusão do gelo é de aproximadamente 334 J/g, e a capacidade térmica da água é de cerca de 4,18 J/g°C.

Resposta 1 : calcule a variação de temperatura:
ΔT = 0°C - (-10°C) = 10°C
Agora, utilize a fórmula Q = mcΔT:
Q = 50 g * 2,1 J/g°C * 10°C = 1.050 J

Respostas

Resposta 2: Primeiro, calcule a quantidade de calor necessária para a fusão do gelo:
Q1 = mL = 200 g * 334 J/g = 66.800 J
Agora, calcule a quantidade de calor necessária para aquecer a água líquida de 0°C para 10°C:
Q2 = mcΔT = 200 g * 4,18 J/g°C * (10°C - 0°C) = 8.360 J
A quantidade total de calor necessária é a soma de Q1 e Q2:
Q total = Q1 + Q2 = 66.800 J + 8.360 J = 75.160 J

Estados da matéria

Pergunta 1: Pergunta 1: Em uma experiência em laboratório, um pesquisador tem um recipiente contendo 1 litro de nitrogênio gasoso à temperatura de -196°C. Se ele quiser liquefazer todo o nitrogênio, quanto calor precisará remover do sistema? O ponto de ebulição do nitrogênio é de -196°C, e o calor latente de liquefação do nitrogênio é de aproximadamente 199 J/g. Considere a densidade do nitrogênio gasoso a -196°C como 0,81 g/L.

Pergunta 2: Pergunta 2: Um recipiente contém 500 ml de água a 25°C. Se essa água for aquecida a uma taxa constante, quanto tempo levará para a água atingir o ponto de ebulição a 100°C? Considere a capacidade térmica da água como 4,18 J/g°C.

calcule a massa de nitrogênio no recipiente:
Massa = Densidade * Volume
Massa = (0,81 g/L) * (1000 mL) = 810 g
Agora, calcule a quantidade de calor necessária para liquefazer todo o nitrogênio:
Q = mL
Q = (810 g) * (199 J/g) = 161.190 J

Respostas

calcule a quantidade de calor necessária para elevar a temperatura da água de 25°C para 100°C:
Q = mcΔT
Q = (500 g) * (4,18 J/g°C) * (100°C - 25°C) = 167.200 J
Agora, considere que a taxa de aquecimento é constante e queremos saber quanto tempo levará para atingir esse calor:
Tempo = Q / Potência
Tempo = 167.200 J / Potência

Transferência de calor

Pergunta 1: Uma barra de ferro de 2 metros de comprimento está inicialmente a uma temperatura de 200°C. Se ela for colocada em uma sala a 20°C, quanto tempo levará para que a barra atinja a temperatura ambiente? A condutividade térmica do ferro é de aproximadamente 80,2 J/(m·s·°C).

Pergunta 2: Um dispositivo eletrônico emite calor a uma taxa constante de 500 Watts. Ele está localizado dentro de um invólucro de isolamento térmico perfeito, exceto por uma única janela de vidro. A temperatura ambiente externa é de 25°C, e a temperatura desejada internamente é de 45°C. Se a área da janela de vidro é de 0,5 m² e o coeficiente de transferência de calor do vidro é de 0,8 W/(m²·°C), quanto tempo levará para que o dispositivo alcance a temperatura interna desejada após ser ligado?

calcule a diferença de temperatura entre a barra de ferro e a temperatura ambiente:
ΔT = 200°C - 20°C = 180°C
Agora, utilize a fórmula do calor por condução para calcular o tempo necessário:
Q = (k * A * ΔT) / d
Q = (80,2 J/(m·s·°C) * 2 m * 180°C) / 1 m = 28.872 J
Tempo = Q / Potência
Tempo = 28.872 J / Potência

Respostas

calcule a diferença de temperatura entre a temperatura interna e a temperatura ambiente:
ΔT = 45°C - 25°C = 20°C
Agora, utilize a fórmula de transferência de calor por radiação:
Potência = A * h * ΔT
Onde h é o coeficiente de transferência de calor do vidro (0,8 W/(m²·°C)) e A é a área da janela de vidro (0,5 m²).
Potência = (0,5 m²) * (0,8 W/(m²·°C)) * 20°C = 8 W
Agora, calcule o tempo necessário:
Tempo = Q / Potência
Tempo = Q / 500 W
Como a potência é constante a 500 Watts:
Tempo = Q / 500 W = 8 W / 500 W = 0,016 horas
0,016 horas * 60 minutos/hora = 0,96 minutos
Portanto, levará aproximadamente 0,96 minutos (ou cerca de 58 segundos) para que o dispositivo alcance a temperatura interna desejada.

isolantes e materiais condutores

Pergunta 1: Você tem duas barras idênticas, uma feita de alumínio e outra de madeira. Ambas as barras estão inicialmente a 20°C. Você coloca ambas em uma sala a 40°C. Qual das barras atingirá uma temperatura mais alta após um período de tempo? Explique sua resposta.

Pergunta 2: Você está projetando um recipiente isolado termicamente para armazenar gelo por um longo período de tempo em um ambiente externo quente. Você tem dois materiais isolantes diferentes para escolher: espuma de poliestireno e fibra de vidro. A espuma de poliestireno tem uma condutividade térmica de 0,03 W/(m·°C), enquanto a fibra de vidro tem uma condutividade térmica de 0,05 W/(m·°C). Qual desses materiais isolantes é mais eficaz para manter o gelo congelado em um ambiente quente? Justifique sua escolha.

A barra de alumínio atingirá uma temperatura mais alta após um período de tempo. Isso ocorre porque o alumínio é um bom condutor de calor, o que significa que ele permite a transferência de calor de forma eficiente. A barra de madeira, por outro lado, é um isolante térmico e não conduz calor tão bem quanto o alumínio. Portanto, a barra de madeira levará mais tempo para aquecer e atingir a temperatura da sala.

Respostas

A espuma de poliestireno é mais eficaz para manter o gelo congelado em um ambiente quente. Isso ocorre porque a espuma de poliestireno tem uma condutividade térmica mais baixa (0,03 W/(m·°C)) em comparação com a fibra de vidro (0,05 W/(m·°C)). Quanto menor a condutividade térmica, melhor o material isolante é em impedir a transferência de calor. Portanto, a espuma de poliestireno reduzirá a transferência de calor do ambiente externo para o interior do recipiente de gelo com mais eficácia do que a fibra de vidro, mantendo o gelo congelado por um período mais longo.

Calorímetros

Pergunta 1: Um estudante de laboratório deseja determinar o calor específico de uma amostra desconhecida. Ele coloca 200 gramas dessa amostra em um calorímetro e adiciona 100 ml de água a 20°C. A temperatura final do sistema após o equilíbrio é de 30°C. Qual é o calor específico da amostra desconhecida? Use a fórmula Q = mcΔT, onde c é o calor específico da amostra desconhecida.

Pergunta 2: Um calorímetro contém 250 ml de água a 25°C. Uma amostra de metal, inicialmente a 100°C e com massa de 50 gramas, é colocada na água do calorímetro. A temperatura final do sistema após o equilíbrio é de 35°C. Qual é o calor específico do metal? Use a fórmula Q = mcΔT, onde c é o calor específico do metal.

calcule a quantidade de calor ganha pela amostra desconhecida:
Q1 = mcΔT = (200 g) * c * (30°C - 20°C)
Agora, calcule a quantidade de calor perdida pela água:
Q2 = mcΔT = (100 g) * (4,18 J/g°C) * (30°C - 20°C)
Como o calor ganho pela amostra é igual ao calor perdido pela água (lei da conservação de energia), podemos igualar Q1 e Q2:
(200 g) * c * (30°C - 20°C) = (100 g) * (4,18 J/g°C) * (30°C - 20°C)
Agora, resolvendo para c:
c = [(100 g) * (4,18 J/g°C) * (30°C - 20°C)] / [(200 g) * (30°C - 20°C)] = 2,09 J/g°C

Respostas

calcule a quantidade de calor ganha pela água:
Q1 = mcΔT = (250 g) * (4,18 J/g°C) * (35°C - 25°C)
Agora, calcule a quantidade de calor perdida pelo metal:
Q2 = mcΔT = (50 g) * c * (35°C - 100°C)
Como o calor ganho pela água é igual ao calor perdido pelo metal (lei da conservação de energia), podemos igualar Q1 e Q2:
(250 g) * (4,18 J/g°C) * (35°C - 25°C) = (50 g) * c * (35°C - 100°C)
Agora, resolvendo para c:
c = [(250 g) * (4,18 J/g°C) * (35°C - 25°C)] / [(50 g) * (35°C - 100°C)] = 0,897 J/g°C

Calor Latente

Pergunta 1: Um estudante deseja determinar o calor latente da fusão do gelo. Ele coloca uma massa de 100 gramas de gelo a -10°C em um calorímetro contendo 200 ml de água a 30°C. Após o equilíbrio térmico, a temperatura final é de 0°C. Qual é o calor latente da fusão do gelo? Use a fórmula Q = mL, onde L é o calor latente de fusão do gelo.

Pergunta 2:Um experimentador deseja determinar o calor latente de vaporização da água. Ele coloca uma massa de 150 gramas de água a 100°C em um calorímetro contendo 50 gramas de gelo a -10°C. Após o equilíbrio térmico, a temperatura final é de 0°C. Qual é o calor latente de vaporização da água? Use a fórmula Q = mL, onde L é o calor latente de vaporização da água.

Calcule a quantidade de calor perdida pela água ao resfriar de 30°C para 0°C:
Q1 = mcΔT = (200 g) * (4,18 J/g°C) * (0°C - 30°C)
Agora, calcule a quantidade de calor ganha pelo gelo ao aquecer de -10°C para 0°C:
Q2 = mcΔT = (100 g) * (4,18 J/g°C) * (0°C - (-10°C))
Como o calor ganho pelo gelo é igual ao calor perdido pela água (lei da conservação de energia), podemos igualar Q1 e Q2:
(200 g) * (4,18 J/g°C) * (0°C - 30°C) = (100 g) * L
Agora, resolvendo para L (calor latente de fusão do gelo):
L = [(200 g) * (4,18 J/g°C) * (0°C - 30°C)] / (100 g) = 25,08 J/g

Respostas

Calcule a quantidade de calor perdida pela água ao resfriar de 100°C para 0°C:
Q1 = mcΔT = (150 g) * (4,18 J/g°C) * (0°C - 100°C)
Agora, calcule a quantidade de calor ganha pelo gelo ao aquecer de -10°C para 0°C:
Q2 = mcΔT = (50 g) * (4,18 J/g°C) * (0°C - (-10°C))
Como o calor ganho pelo gelo é igual ao calor perdido pela água (lei da conservação de energia), podemos igualar Q1 e Q2:
(150 g) * (4,18 J/g°C) * (0°C - 100°C) = (50 g) * L
Agora, resolvendo para L (calor latente de vaporização da água):
L = [(150 g) * (4,18 J/g°C) * (0°C - 100°C)] / (50 g) = 125,4 J/g

Calor específico

Pergunta 1: (UNIP-SP - ADAPTADA) O calor específico latente de fusão do gelo é de 80 cal/g. Para fundir uma massa de gelo de 80g, sem variação de temperatura, a quantidade de calor latente necessária é de:
a)1,0 cal b) 6,4 cal c ) 1000,0 cal d) 64000 c a l e) 6,4 x 10^3 cal

Pergunta 2: (FUVEST - ADAPTADA) Dispõe-se de água a 80°C e gelo a 0°C. Deseja-se obter 100 gramas de água a uma temperatura de 40°C (após o equilíbrio), misturando água e gelo em um recipiente isolante e com capacidade térmica desprezível. Sabe-se que o calor específico latente de fusão do gelo é 80 cal/g e o calor específico sensível da água é 1,0 cal/g°C. A massa de gelo a ser utilizada é:

O calor latente é o produto entre a massa e o calor específico.
Q = M x l
Q = 80 x 80 = 6400 cal
Escrevendo em notação científica
Q = 6,4 x 10^3 cal

Respostas

Equação 1:
mágua + mgelo = 100g
Modelo matemático a ser usado:
No equilíbrio, as transferências de calor serão iguais a zero.
Qgelo + Qágua = 0 (equação 2)
O gelo passa por dois processos: o de fusão a 0°C e o de aquecimento da fase líquida de 0°C a 40°C.
Qgelo = mgelo * L + mgelo * c * Δθ
A água líquida passa por uma redução de temperatura, neste processo, apenas o calor sensível age.
Qágua = mágua * c * Δθ
Substituindo os calores da água e do gelo na equação 2, temos:
mgelo * L + mgelo * c * Δθ + mágua * c * Δθ = 0
Substituindo os valores fornecidos nos dados:
mgelo * 80 + mgelo * 1 * (40 - 0) + mágua * 1 * (40 - 80) = 0
Simplificando a equação:
80mgelo + 40mgelo - 40mágua = 0
120mgelo - 40mágua = 0
Isolando mgelo:
120mgelo = 40mágua
mgelo = 40mágua / 120
mgelo = (1/3)mágua
Determinamos que a massa de gelo é um terço da massa de água líquida.
Substituindo essa relação na equação 1:
mágua + mgelo = 100g
mágua + (1/3)mágua = 100g
(4/3)mágua = 100g
Isolando mágua:
mágua = (3/4) * 100g
mágua = 75g
Portanto, a massa de água líquida é de 75g, e a massa de gelo é de 25g.
