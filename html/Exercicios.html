Fusão de metais

Pergunta 1: O ferro é um metal amplamente utilizado em construções e na fabricação de veículos. Qual é a quantidade de calor necessária para aquecer 500 g de ferro a partir da temperatura ambiente (25°C) até sua temperatura de fusão (aproximadamente 1.538°C)? Utilize a fórmula Q = mcΔT, onde a capacidade térmica do ferro é de cerca de 0,45 J/g°C. Dica: comece calculando a variação de temperatura.

Pergunta 2: Uma barra de alumínio com massa de 1.000 gramas é aquecida até sua temperatura de fusão de aproximadamente 660,3°C. Se a barra estava inicialmente a 20°C, quanto calor foi fornecido para realizar essa fusão? Utilize a fórmula Q = mL, onde o calor latente de fusão do alumínio é de cerca de 397 J/g. Dica: comece calculando a quantidade de calor necessária para aquecer o alumínio até a temperatura de fusão

Resposta 1 : Primeiro, calcule a variação de temperatura:
ΔT = 1.538°C - 25°C = 1.513°C
Agora, utilize a fórmula Q = mcΔT:
Q = 500g * 0,45 J/g°C * 1.513°C = 340.425 J

Respostas

Resposta 2: Primeiro, calcule a quantidade de calor necessária para aquecer o alumínio até sua temperatura de fusão:
Q1 = mcΔT = 1000 g* 0,9 J/g°C * (660,3°C - 20°C) = 532.629 J
Agora, calcule aquantidade de calor necessária para a fusão do alumínio:
Q2 = mL =1000 g * 397 J/g = 397.000 J
A quantidade total de calor fornecida é a soma de Q1 e Q2:
Q total = Q1 + Q2 = 532.629 J + 397.000 J = 929.629 J

Fusão de gelo

Pergunta 1: Você tem um cubo de gelo de 50 gramas a -10°C. Quanta energia calorífica é necessária para elevar a temperatura do gelo até 0°C? Utilize a fórmula Q = mcΔT, onde a capacidade térmica do gelo é de cerca de 2,1 J/g°C.

Pergunta 2: Uma pessoa tem uma placa de gelo de 200 gramas a -20°C e deseja transformá-la completamente em água líquida a 10°C. Qual a quantidade total de calor necessária para realizar essa transformação? Utilize a fórmula Q = mL para o processo de fusão e Q = mcΔT para aquecer a água líquida até 10°C. O calor latente da fusão do gelo é de aproximadamente 334 J/g, e a capacidade térmica da água é de cerca de 4,18 J/g°C.

Resposta 1 : calcule a variação de temperatura:
ΔT = 0°C - (-10°C) = 10°C
Agora, utilize a fórmula Q = mcΔT:
Q = 50 g * 2,1 J/g°C * 10°C = 1.050 J

Respostas

Resposta 2: Primeiro, calcule a quantidade de calor necessária para a fusão do gelo: Q1 = mL = 200 g * 334 J/g = 66.800 J Agora, calcule a quantidade de calor necessária para aquecer a água líquida de 0°C para 10°C: Q2 = mcΔT = 200 g * 4,18 J/g°C * (10°C - 0°C) = 8.360 J A quantidade total de calor necessária é a soma de Q1 e Q2: Q total = Q1 + Q2 = 66.800 J + 8.360 J = 75.160 J
